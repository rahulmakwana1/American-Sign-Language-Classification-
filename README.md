# American-Sign-Language-Classification

A simple CNN project for detecting american sign language. Here, I have implemented CNN (Convolution Neural Network) using Keras.

Tools Used

1)Python 3




2)OpenCV2



3)Tensorflow



4)Keras


Running this project
## Install Python 3, Opencv 3, Tensorflow, Keras.

1)Run our American sign language classification.ipynb file.



2)run recog.py file to detect gesture in real time.
